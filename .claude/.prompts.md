## Init

我需要一个可以阅读、翻译、转换 Agentic AI 研究领域学术论文的 Agents 工程，用于收集和翻译人工智能智能体领域的重要研究论文，同时提供了辅助阅读这些论文的 Agents 和平台，为中文读者提供高质量的技术资源。

首先，我需要一个可以高质量处理 PDF/Web 格式 Paper/Article 的 Agents，用于将人工智能智能体领域的重要研究论文翻译成中文并转换成 Markdown 格式进行本地化。这些 Agents 放在 `agents/` 路径下，其中使用 claude-agent-sdk 搭建的 agents 放于 `agents/claude/` 路径下；使用 google-adk 搭建的 agents 放于 `agents/adk/` 路径下。
其次，我需要一个可以用于管理这些 PDF/Web 格式 Paper/Article 及其转换文档的 Admin UI 系统，用于在 Web UI 端管理文档和实施文档转换工作。这个 Admin UI 系统放在 `ui/` 路径下。
最后，我需要直接在本项目仓库存放这些 PDF/Web 格式 Paper/Article 及其转换文档，它们统一放在 `papers/` 路径下，其中源文档在 `papers/source/`、翻译文档在 `papers/translation`、愫读文档在 `papers/heartfelt`。

帮我规范化这个项目仓库。
