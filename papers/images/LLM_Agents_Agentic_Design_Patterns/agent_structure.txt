This directory will contain extracted figures from the Agentic Design Patterns paper.

Figure 1: An agentic system primarily consists of an LLM core, thought processes (e.g., prompting, reflection), and toolsets (e.g., web search, code execution). It interacts with users via interfaces and acts upon external tools through a tooling layer. This paper provides a comprehensive survey of agentic design patterns that enable intelligent agent applications.

图中显示了一个典型的智能体系统架构。中央核心是大型语言模型（LLM），周围环绕着三个主要组件：
1. 思考过程（Thought Process）- 包括提示（Prompting）、反思（Reflection）、规划（Planning）、记忆（Memory）等
2. 工具集（Toolset）- 包括网页搜索（Web Search）、代码执行（Code Execution）、LLM工具（LLM Tools）、计算器（Calculator）等
3. 用户界面（User Interface）- 通过接口与用户交互

整个系统通过一个工具层（Tooling Layer）与外部工具进行交互，实现各种智能应用。
